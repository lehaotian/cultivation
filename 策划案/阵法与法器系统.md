# 阵法与法器系统

## 概述

阵法和法器是"自动化流水线玩法"的核心承载，将灵力的被动流转变为可配置、可优化、可扩产的生产线。

**核心理念**：阵法和法器不是一次性消耗品，而是可以持续投入、升级、组合的"灵力装备"。

---

## 一、阵法（Formations）

阵法是静态的灵力处理管道，布置后持续生效，像一条"灵力流水线"。

### 阵法结构

```
[灵气输入] → [阵基] → [阵眼] → [效果输出]
                 ↑
              [阵旗/材料]
```

| 组件 | 说明 | 可升级 |
|:----|:-----|:------:|
| **阵基** | 阵法骨架，决定了阵法的基础类型 | ✅ 消耗材料加固 |
| **阵眼** | 阵法核心，决定了阵法的效果强度 | ✅ 投入灵力升级 |
| **阵旗** | 辅助组件，可插入多个提供附加效果 | ✅ 可替换/强化 |
| **材料** | 维持阵法运转的消耗品 | 持续消耗 |

### 阵法类型

| 类型 | 功能 | 玩法类比 |
|:----|:-----|:---------|
| **聚灵阵** | 提升区域灵气浓度 | 资源生产建筑 |
| **护山大阵** | 防御敌方入侵 | 防御塔/城墙 |
| **传送阵** | 快速移动/跨区传送 | 传送门网络 |
| **迷阵** | 减缓/误导敌方行动 | 减速/控制区域 |
| **炼器阵** | 自动生产法器 | 制造工厂 |
| **炼丹阵** | 自动生产丹药 | 炼药流水线 |
| **淬体阵** | 提升弟子修炼速度 | 经验加成 |
| **困敌阵** | 困住进入的敌人 | 囚笼/陷阱 |

### 阵法自动化流水线

阵法可以串联形成自动化流水线：

```
聚灵阵 → 炼器阵 → 成品法器
   ↑         ↓
 灵脉    材料输入
```

- 玩家只需搭建好阵法链，填入材料和灵力
- 阵法会自动运转产出
- 可通过优化阵法链中的每个环节来提高效率

### 阵法升级路径

```
凡阵 → 灵阵 → 玄阵 → 地阵 → 天阵 → 道阵
```

每升一级需要：阵基材料 + 灵力注入 + 对应品阶的阵法师

## 二、法器（Artifacts）

法器是动态的灵力处理单元，可由玩家操控或配置自动策略。

### 法器属性

| 属性 | 说明 |
|:----|:-----|
| **品阶** | 凡→灵→玄→地→天→道（六阶） |
| **属性** | 金木水火土无（与功法对应） |
| **灵容量** | 可储存的灵力上限 |
| **吸收率** | 法器自动吸收灵气的效率 |
| **转化率** | 灵力→效果的转化效率 |
| **耐久** | 使用后损耗，可修复 |
| **槽位** | 可镶嵌灵石/符文的数量 |

### 法器类型

| 类型 | 功能 | 场景 |
|:----|:-----|:-----|
| **飞剑** | 攻击型 | 战斗、征伐 |
| **法袍** | 防御型 | 减伤、保护 |
| **灵珠** | 储存型 | 灵力银行 |
| **阵盘** | 布阵型 | 快速部署阵法 |
| **符箓** | 消耗型 | 一次性强力法术 |
| **法宝** | 全能型 | 核心装备 |

### 法器自动化

- 法器可配置**自动行为**（如：灵力满后自动释放法术）
- 法器可以接入阵法流水线作为节点
- 法器之间可组合形成**法器连携**（类似套装效果）

---

## 三、阵法+法器组合示例

```
[护山大阵] + [飞剑阵眼] + [灵石填充]
     ↓
自动攻击进入范围的敌人（类似塔防）

[聚灵阵] → [炼器阵] + [五行灵石]
     ↓
自动生产对应属性的法器

[传送阵网络] + [灵珠中继站]
     ↓
组织领地内快速反应
```
