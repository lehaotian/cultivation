# 联盟与道统系统

## 概述

联盟是组织之上的更高层结构，称为**道统**。不同的道统代表着对"天道"的不同理解与追求。加入道统意味着认同其理念，并为之奋斗。

一个组织在同一时间只能加入一个道统。道统之间是竞争与合作并存的关系。

## 三大道统

### 司天 —— 号令天道，定数安排

> *"天道有序，万物皆可算。吾等代天行事，定天下格局。"*

| 项目 | 内容 |
|:----|:-----|
| **难度** | ⭐⭐⭐⭐⭐（最难） |
| **收益** | ⭐⭐⭐⭐⭐（最大） |
| **核心理念** | 掌握天道规律，通过推演预知未来，安排世间一切 |
| **终极目标** | 获得号令天下的权限，成为天道代言人 |

**特色机制：**

- **天机推演**：可消耗资源推演未来事件（灵气潮汐、资源刷新等）
- **道令**：可发布全服通告/任务，影响力极高
- **定数安排**：可一定程度影响游戏世界的事件走向
- **代价**：窥探天机需要献祭大量资源/功勋，且每次推演都有反噬风险

**盟主特权：**
- 可发布全服"天道令"（所有组织收到）
- 可查看一部分其他势力的情报
- 可调配中立区域的资源分配

### 幽冥 —— 往生转世，轮回不息

> *"生死之间，方见真我。轮回不休，大道不绝。"*

| 项目 | 内容 |
|:----|:-----|
| **难度** | ⭐⭐⭐⭐ |
| **收益** | ⭐⭐⭐⭐ |
| **核心理念** | 掌握生死轮回之力，借往生之道壮大自身 |
| **终极目标** | 参透生死，超脱轮回 |

**特色机制：**

- **往生转世**：组织成员战死后可保留部分修为转世，而不是完全损失
- **灵魂收割**：可在凡人城镇收集灵魂资源，用于强化或复活
- **幽冥通道**：可建立连接各地幽冥通道，实现快速转移
- **轮回buff**：周期性的轮回增益，契合灵气潮汐的节奏

**独特玩法：**
- 不怕战损，适合激进的作战风格
- 转世后组织ID不变但位置可能重置，形成策略性
- 可以"收容"其他组织战败后的残余力量

### 真武 —— 求真务实，以力证道

> *"大道至简，唯力不破。求真务实，以战证道。"*

| 项目 | 内容 |
|:----|:-----|
| **难度** | ⭐⭐⭐ |
| **收益** | ⭐⭐⭐ |
| **核心理念** | 通过战斗征服求真，力量即真理 |
| **终极目标** | 统一天下，以绝对武力证明道之真谛 |

**特色机制：**

- **武运加持**：战斗胜利越多，全组织战力加成越高
- **征服体系**：占领的领地提供额外收益
- **战意系统**：连续作战可获得愈战愈勇的增益
- **武道塔**：PVE挑战玩法，层数越高解锁越强能力

**独特玩法：**
- 正面对抗最强，适合PVP核心玩家
- 领地扩张收益递增，但树敌也多
- 战败惩罚更重（真武不允许退缩）

## 道统对比

| 对比维度 | 司天 | 幽冥 | 真武 |
|:--------|:----:|:----:|:----:|
| 上手难度 | 极高 | 中 | 低 |
| PVP强度 | 中 | 高 | 极高 |
| 生存能力 | 中 | 极高 | 低 |
| 资源需求 | 极高 | 中 | 低 |
| 盟主收益 | 极高 | 高 | 中 |
| 适合人群 | 策略型 | 运营型 | 战斗型 |

## 加入/退出道统

- 组织创建时可选择倾向（不绑定）
- 达到一定等级后可正式加入道统
- 退出道统有冷却期和惩罚（部分能力暂时失效）
- 叛逃到敌对道统惩罚更重（可能触发追杀机制）
