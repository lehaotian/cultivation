# 灵气潮汐与轮回机制

## 核心概念：四服轮回

设定存在4个服务器（4界），每个服务器独立运行但遵循同一套轮回周期。**4周一个完整轮回**，每服错峰，形成跨服战略窗口。

### 为什么是4服4周？

```
服1: 涨潮 → 峰值 → 退潮 → 谷底 | 涨潮 → ...
服2: 谷底 | 涨潮 → 峰值 → 退潮 | 谷底 → ...
服3: 退潮 | 谷底 | 涨潮 → 峰值 | 退潮 → ...
服4: 峰值 → 退潮 | 谷底 | 涨潮 | 峰值 → ...
     ↑--- 第1周 ---↑--- 第2周 ---↑--- 第3周 ---↑--- 第4周 ---↑
```

- 4服灵气潮汐相位各差1周
- 每服每4周完成一次完整潮汐周期
- 因此每服一年经历13次灵气轮回（52周 ÷ 4）

### 灵气潮汐四阶段

| 阶段 | 持续时间 | 灵气倍率 | 特征 |
|:----|:--------:|:--------:|:----|
| **涨潮期** 🌊 | 1周 | ×1.0→×3.0（渐增） | 灵气回升，宜发展 |
| **峰值期** 🌕 | 1周 | ×3.0（稳定） | **黄金周**，宜突破/大战 |
| **退潮期** 📉 | 1周 | ×3.0→×1.0（渐减） | 灵气下降，宜收割/收敛 |
| **谷底期** 🌑 | 1周 | ×0.5（最低） | **衰弱期**，消耗增加，宜休整 |

## 二、灵气潮汐的影响

### 正面影响（涨潮→峰值）

| 系统 | 效果 |
|:----|:-----|
| **修炼速度** | 基础修炼速度 × 潮汐系数 |
| **阵法效率** | 聚灵阵产出 × 潮汐系数 |
| **法术威力** | 法术伤害 × √潮汐系数 |
| **资源刷新** | 刷新量 × 潮汐系数 |
| **突破成功率** | 突破概率 + 潮汐系数修正 |

### 负面影响（退潮→谷底）

| 系统 | 效果 |
|:----|:-----|
| **修炼速度** | 降低至50% |
| **阵法维持费** | 增加50% |
| **经脉损耗** | 增加30% |
| **突破成功率** | 降低20% |
| **灵池衰减** | 加速 |

## 三、跨服交互：潮汐窗口

由于4服潮汐错峰，就出现了"此消彼长"的战略窗口：

| 你的服务器状态 | 对服状态 | 战略建议 |
|:-------------|:--------|:---------|
| **峰值期** | 谷底期 | 🔥 最佳入侵窗口！跨服掠夺 |
| **峰值期** | 峰值期 | ⚔️ 公平对决 |
| **谷底期** | 峰值期 | 🛡️ 防守模式，龟缩待机 |
| **谷底期** | 谷底期 | 🌱 各自休养生息 |

### 跨服玩法（规划）

- 跨服通道仅在特定潮汐阶段开放
- 在潮汐优势期入侵弱服可获得额外收益
- 被入侵方在自服有防守加成（主场优势）
- 跨服战果影响轮回结算

## 四、轮回结算

每4周（一次完整轮回）结束时进行**轮回结算**：

### 结算内容

1. **道统贡献排名**
   - 根据轮回内的综合表现（资源、战斗、发展）排名
   - 排名奖励：天道功德、特殊资源、称号

2. **灵气重置**
   - 灵脉部分重新分布
   - 秘境刷新
   - 世界事件重置

3. **轮回奖励**
   - 参与度奖励（活跃组织）
   - 道统目标完成度奖励

4. **跨服排名**
   - 4服统一排名
   - 全服前N名获得跨界特权

### 长期轮回

- 每4周为一小轮回
- 每4小轮回（16周）为一中轮回
- 每4中轮回（64周≈1年）为一大轮回
- 大轮回触发全服重大事件

## 五、潮汐策略价值

```
涨潮期（第1周）：扩张领地、采集资源、升级建筑
峰值期（第2周）：突破境界、发动战争、挑战秘境
退潮期（第3周）：巩固现有成果、处理战利品、合约外交
谷底期（第4周）：休整、内政、储备资源、策划下一轮
```

### 道统与潮汐策略差异

| 道统 | 潮汐策略 |
|:----|:---------|
| **司天** | 精准预测潮汐窗口，提前布局，在峰值期发动最优进攻 |
| **幽冥** | 谷底期反而是优势期（死亡转世在低潮时惩罚更低） |
| **真武** | 峰值期最强，追求在黄金周内决出胜负 |
