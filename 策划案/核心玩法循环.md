# 核心玩法循环

## 整体循环

```
                     ┌──────────────────────┐
                     │     创建组织          │
                     │ 选择性质(门/派/教/宗) │
                     └──────────┬───────────┘
                                ▼
                     ┌──────────────────────┐
                     │   发展期（涨潮期）    │
                     │ · 建设山门/传播教义  │
                     │ · 采集资源           │
                     │ · 打通经脉节点       │
                     │ · 构筑阵法流水线     │
                     └──────────┬───────────┘
                                ▼
                     ┌──────────────────────┐
                     │   繁荣期（峰值期）    │
                     │ · 突破境界           │
                     │ · 攻占资源点         │
                     │ · 道统战争           │
                     │ · 挑战秘境           │
                     └──────────┬───────────┘
                                ▼
                     ┌──────────────────────┐
                     │   收缩期（退潮期）    │
                     │ · 巩固领地           │
                     │ · 调整阵法配置       │
                     │ · 外交合作           │
                     └──────────┬───────────┘
                                ▼
                     ┌──────────────────────┐
                     │   蛰伏期（谷底期）    │
                     │ · 储备资源           │
                     │ · 研究功法           │
                     │ · 跨服/内政          │
                     │ · 规划下一轮回       │
                     └──────────┬───────────┘
                                ▼
                     ┌──────────────────────┐
                     │   轮回结算           │
                     │ · 道统贡献排名       │
                     │ · 灵气重置           │
                     │ · 奖励发放           │
                     └──────────┬───────────┘
                                ▼
                     ┌──────────────────────┐
                     │    进入下一轮回       │
                     └──────────────────────┘
```

## 二、玩家日/周常循环

### 日常行为

| 行为 | 频率 | 说明 |
|:----|:----|:-----|
| **收菜** | 每日 | 收取阵法产出、城镇税收 |
| **修炼指令** | 每日 | 下达弟子的修炼方向 |
| **采集指令** | 每日 | 派遣弟子采集资源点 |
| **巡视领地** | 每日 | 查看领地状态，处理事件 |
| **市场交易** | 每日 | 与其他组织交易资源 |

### 周期行为

| 行为 | 频率 | 说明 |
|:----|:----|:-----|
| **秘境挑战** | 每周 | 限时副本，获取特殊资源 |
| **道统任务** | 每周 | 完成道统发布的任务 |
| **攻城战** | 周期 | 大型PVP活动（潮汐峰值期） |
| **跨服活动** | 周期 | 跨服交互玩法 |

### 轮回行为

| 行为 | 频率 | 说明 |
|:----|:----|:-----|
| **道统目标** | 每轮回 | 选择本轮道统发展方向 |
| **灵气重置规划** | 每轮回 | 根据新灵气分布调整策略 |
| **重大决策** | 每轮回 | 组织发展路径选择 |

## 四、胜利条件（道统目标）

### 司天 —— 天机推演完成度

- 累计天机推演达到100%
- 每轮回可推演部分天机
- 当所有天机全部推演完成 → 号令天下

### 幽冥 —— 转世次数/灵魂总量

- 累计转世达到一定次数
- 累计收集灵魂达到阈值
- 当轮回之力足够 → 超脱轮回

### 真武 —— 统一进度

- 占领地图一定比例
- 击败所有敌对道统达到一定次数
- 当武力足够 → 以力证道

### 通用目标

- 组织等级、修炼境界、阵法等级、法器收集度等
- 综合评分排名

## 五、失败的代价

| 情景 | 损失 | 后续 |
|:----|:-----|:-----|
| 山门被破 | 大量资源、部分领地 | 可重建/寻找新驻地 |
| 组织解散 | 基础资产清零 | 可用幽冥道统转世部分资产 |
| 轮回后重置 | 灵气分布重置，但修为/道统贡献保留 | 重新规划 |
| 彻底覆灭 | 完全失败 | 可重新创建组织（重新开始） |
