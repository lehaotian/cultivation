# 修炼体系（功法 / 法术）

## 核心概念：灵气→功法→经脉→法术

修炼体系的核心逻辑是一条完整的灵力流转链路：

```
    吸收             转化             释放
灵气 ──→ [功法] ──→ [经脉] ──→ [法术/阵法/法器]
           ↑              ↑              ↑
        吸取率        转化效率         利用率
```

---

## 一、功法（Cultivation Techniques）

功法是组织吸收灵气的基础能力，决定了灵气**输入**的效率和质量。

### 功法属性

| 属性 | 说明 |
|:----|:-----|
| **品阶** | 凡→灵→玄→地→天→道（六阶） |
| **属性** | 金、木、水、火、土、无（六种） |
| **吸取率** | 功法从环境中吸取灵气的效率（%） |
| **转化率** | 吸入灵气转化为可用灵力的效率（%） |
| **经脉负荷** | 运行时对经脉节点的压力大小 |
| **兼容性** | 与不同类型的法术/法器的配合度 |

### 阴阳五行 · 功法体系

功法按**五行**属性划分，每种属性有独特的运行规律：

| 属性 | 特点 | 适合场景 |
|:----|:----|:---------|
| **金** | 锋锐、攻坚、破防 | 战斗、破坏性法术 |
| **木** | 生长、恢复、缠绕 | 疗伤、控制、资源产出 |
| **水** | 柔韧、变化、渗透 | 防御、持久战、信息 |
| **火** | 爆发、炽烈、毁灭 | 强攻、群伤、炼器 |
| **土** | 厚重、稳固、承载 | 防御、阵法、资源存储 |
| **无** | 中性、兼容、调和 | 辅助、平衡、特殊 |

### 经脉节点（创新点）

人体/组织有**经脉节点网络**，功法的运行需要打通特定节点：

- 每个功法需要激活特定的经脉节点组合（类似于技能树/符文盘）
- 节点之间的**连通性**决定了灵气流经的效率
- 打通更多节点 → 可同时运行更多功法/法术
- 节点之间的连接方式影响整体灵力循环的质量

**节点类型：**

| 节点类型 | 作用 |
|:--------|:-----|
| 主脉 | 主干道，决定了灵力上限 |
| 支脉 | 分支，决定了特定属性功法的效率 |
| 窍穴 | 特殊节点，打通后解锁特殊能力 |
| 关隘 | 瓶颈，需要突破才能继续发展 |

## 二、法术（Spells）

法术是将经脉中流转的灵力释放到外界的最终体现。本质上是**经脉与外界灵气的链接**。

### 法术构成

```
法术 = 灵力输入 × 法术模板 × 经脉适配度
```

- **灵力输入**：由功法提供，决定了法术的基础威力
- **法术模板**：法术本身的架构（决定了灵力到效果的转换公式）
- **经脉适配度**：组织的经脉节点网络与该法术的匹配程度

### 法术分类

| 类型 | 特点 | 例子 |
|:----|:----|:-----|
| **攻击型** | 直接造成伤害 | 火球术、金刃术 |
| **防御型** | 减少受到的伤害 | 土盾术、水幕术 |
| **控制型** | 限制敌方行动 | 木缠术、冰封术 |
| **辅助型** | 增益友方 | 回春术、金刚咒 |
| **特殊型** | 改变战局 | 传送阵、幻术 |

### 法术等级

| 等级 | 名称 | 解锁条件 |
|:----|:----|:---------|
| 1 | 入门法术 | 基础经脉激活 |
| 2 | 进阶法术 | 打通特定支脉 |
| 3 | 秘传法术 | 解锁特定窍穴 |
| 4 | 失传法术 | 特殊奇遇/遗迹 |
| 5 | 自创法术 | 悟道+资源投入 |

## 三、灵气流转数值模型（框架）

```
单次修炼产出 = 环境灵气浓度 × 功法吸取率 × 转化效率 × 经脉连通度

环境灵气浓度   = f(地图位置, 灵气潮汐阶段, 五行属性)
功法吸取率     = f(功法品阶, 功法属性匹配)
转化效率       = f(功法自身转化率, 经脉适配度)
经脉连通度     = f(已激活节点数, 节点连接质量)
```

> 具体数值公式见 [数值表] 目录

## 四、修炼方式

| 方式 | 速度 | 消耗 | 风险 |
|:----|:----:|:----:|:----:|
| 静修 | 慢 | 低 | 无 |
| 丹药辅助 | 中 | 中 | 低 |
| 灵气灌顶 | 快 | 高 | 中（经脉损伤风险） |
| 双修/合修 | 中 | 中 | 低（需其他组织配合） |
| 秘境修炼 | 极快 | 极高 | 高（秘境危险） |
