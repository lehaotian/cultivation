# 地图与资源系统

## 一、世界地图架构

修仙世界分多个**区域（地图块）**，每个区域具有不同的属性。

### 区域类型

| 类型 | 说明 | 占据方 |
|:----|:-----|:------|
| **灵山福地** | 灵气充沛，适合山门驻地 | 组织占领 |
| **凡人城镇** | 有人口/灵魂资源 | 中立/争夺 |
| **荒野** | 低灵气区，刷新资源点 | 公共 |
| **秘境** | 特殊副本区域，定时开启 | 公共/争夺 |
| **禁区** | 高危险高回报 | 探索 |

### 地图基础参数

```
地图尺寸：建议 200×200 ~ 500×500 格（待定）
每格类型：平地、山地、水域、森林、荒漠、灵脉等
每格属性：五行属性、灵气浓度、资源类型
```

## 二、凡人城镇

凡人城镇是修仙世界的基础资源来源。

### 城镇属性

| 属性 | 说明 |
|:----|:-----|
| **人口** | 凡人数量，决定了劳动力/信仰来源 |
| **灵魂** | 可用于幽冥道统的特殊资源 |
| **信仰** | 对组织/道统的认同度（影响教派发展） |
| **产出** | 基础材料、粮食、矿石等 |
| **繁荣度** | 影响产出效率 |

### 城镇交互

| 操作 | 效果 | 消耗 |
|:----|:-----|:----|
| **庇护** | 保护城镇，获得税收 | 驻军 |
| **教化** | 传播教义/文化，提升信仰 | 修士时间 |
| **征召** | 从城镇招募弟子/劳动力 | 消耗民心 |
| **掠夺** | 暴力获取资源 | 降低繁荣度，树敌 |
| **净化** | 清除魔气/邪祟 | 灵力 |

### 城镇控制权

- 城镇可被多个组织同时施加影响（竞争）
- 影响力最高的组织获得主要收益
- 教派可通过信仰传播获得天然优势

## 三、灵气节点

地图上定期刷新的高价值资源点。

### 五行灵气节点

| 属性 | 颜色标识 | 产出效果 |
|:----|:--------|:---------|
| 金 | 白色 | 产出金系灵气，适合炼器 |
| 木 | 青色 | 产出木系灵气，适合丹药 |
| 水 | 蓝色 | 产出水系灵气，适合防御 |
| 火 | 红色 | 产出火系灵气，适合攻击 |
| 土 | 黄色 | 产出土系灵气，适合阵法 |

**刷新规律：**
- 每24小时（或每个周期）刷新一批
- 品质随机：凡品~天品不等
- 位置随机，但有一定规律（如山麓多金、水域多水）
- 可被聚灵阵吸附，移动节点位置

### 五行精矿点

| 类型 | 作用 |
|:----|:-----|
| **精金矿** | 炼制金属性法器/阵基 |
| **灵木心** | 木属性建筑/丹药核心材料 |
| **水精石** | 储藏灵力的容器材料 |
| **火晶石** | 阵法能源核心 |
| **土灵玉** | 防御型阵法材料 |

**刷新规律：**
- 刷新速度慢于灵气节点（每3天）
- 品质固定，但矿脉有"储量"概念
- 采集需要时间，可被争夺
- 矿脉枯竭后需要寻找新矿源
