# 灵气系统

## 核心公式：出÷入

灵气系统的核心在于**进入量**与**流出量**的比值，最终体现在利用率上。

```
灵气利用率 = 有效输出 ÷ 总吸收量
总吸收量   = 各功法吸收灵气之和
有效输出   = 总转化量 - 经脉损耗 - 储存衰减

效率链：
吸取率 → 转化率 → 经脉连通度 → 利用率 → 有效输出
```

---

## 一、灵气进入（吸收）

组织吸收灵气有多个渠道：

| 渠道 | 效率 | 稳定性 | 限制 |
|:----|:----:|:------:|:----|
| **地盘灵气** | 中 | 稳定 | 依赖领地灵气浓度 |
| **聚灵阵** | 高 | 稳定 | 需消耗材料和灵力维护 |
| **灵气节点** | 极高 | 周期性 | 随时间衰减刷新 |
| **丹药** | 极高 | 即时 | 有丹毒上限 |
| **掠夺** | 极高 | 一次性 | 需攻打敌对领地 |
| **秘境** | 爆发 | 不稳定 | 秘境开启窗口 |

### 地盘灵气浓度公式（框架）

```
区域灵气浓度 = 基础浓度 × 潮汐系数 × 五行匹配系数 × 建筑加成

基础浓度     = f(地图区域, 灵脉密度)
潮汐系数     = f(当前潮汐阶段)
五行匹配     = f(该区域五行属性 × 功法属性)
建筑加成     = f(聚灵阵等建筑)
```

---

## 二、灵气流出（消耗）

灵气消耗的主要途径：

| 途经 | 占比预估 | 可控性 |
|:----|:-------:|:------:|
| **修炼突破** | 30% | 可控 |
| **法术释放** | 25% | 部分可控 |
| **阵法维持** | 20% | 可控 |
| **法器充能** | 10% | 可控 |
| **炼丹炼器** | 10% | 可控 |
| **经脉自然损耗** | 5% | 不可控（基础损耗） |

### 经脉损耗

- 灵气在经脉中流转时会自然逸散
- 损耗率 = 基础损耗率 × (1 - 经脉连通度)
- 经脉连通度越高，损耗越低
- 可通过特定功法/丹药降低损耗

### 存储衰减

- 储存的灵力会随时间缓慢衰减
- 衰减率 = 基础衰减率 × (1 - 储存品质)
- 高品质储灵设施（灵池、灵珠）可降低衰减
- 极限：天品储灵设施可近乎无衰减

## 三、利用率计算

```
利用率 = 有效输出 ÷ 总吸收量
        = (总吸收量 - 经脉损耗 - 储存衰减) ÷ 总吸收量

举例：
总吸收量 = 1000 灵力/周期
经脉损耗 = 10 (连通度80%时，基础损耗5% × (1-0.8) × 1000 = 10)
储存衰减 = 30 (部分储存未及时使用)
有效输出 = 1000 - 10 - 30 = 960
利用率   = 960 / 1000 = 96%
```

**不同流派的利用率目标：**

| 流派 | 目标利用率 | 策略 |
|:----|:---------:|:-----|
| 爆发型（火） | 追求瞬间高输出，接受低利用率 | 囤积→一波流 |
| 持续型（水） | 长期稳定高利用率 | 细水长流 |
| 均衡型（土） | 稳定中等利用率 | 全面均衡 |

## 四、灵气品质

灵气不仅有"量"，还有"质"：

| 品质 | 颜色 | 效果倍率 | 获取方式 |
|:----|:----|:--------:|:--------|
| 凡品 | 白 | ×1.0 | 普通区域 |
| 灵品 | 绿 | ×1.5 | 灵脉区域 |
| 玄品 | 蓝 | ×2.0 | 灵气节点 |
| 地品 | 紫 | ×3.0 | 秘境、宝地 |
| 天品 | 金 | ×5.0 | 天道馈赠 |

```math
有效灵力 = 灵气量 × 品质倍率 × 利用率
```

## 五、灵气平衡原则

1. **总量守恒**：全服灵气总量约等于全服消耗总量 + 逸散量
2. **零和博弈**：一方多占灵气=其他方少得灵气
3. **周期性重置**：灵气潮汐周期结束后，灵气分布重置
4. **马太效应抑制**：越强的势力经脉负载越高，边际收益递减
5. **五行相生相克**：五行匹配度影响吸收效率，促使多样性发展
