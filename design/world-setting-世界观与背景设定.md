# 世界观与背景设定

## 世界格局

天地初开，鸿蒙分判。灵气充盈于天地之间，万物皆可修行。然天道有常，灵气非永恒不变，而是遵循**灵气潮汐**之律，周而复始、涨落轮回。

- 四界并行，互不相通但同源同根
- 每界（即每服）经历独立灵气轮回
- 四界轮回错峰，形成跨服战略窗口

## 核心矛盾

修仙之路，资源有限而求道者无穷。各方势力为争夺**灵气节点、五行矿脉、凡人信仰**，展开永不休止的博弈。

- **天道之争**：谁号令天道，谁主宰苍生
- **道统之争**：不同理念的流派，追求各自的终极目标
- **资源之争**：灵气、矿脉、人口、灵魂，皆为战略资源

## 灵力本质

灵力并非抽象数值，而是一套完整的**输入-转化-输出**体系：

```
外界灵气 → 功法吸收 → 经脉流转 → 法术释放/阵法驱动/法器催动
     ↑          ↑            ↑              ↑
  灵气浓度   吸收率      转化效率        利用率
```

任何环节的效率提升，都能让整体实力产生质变。
