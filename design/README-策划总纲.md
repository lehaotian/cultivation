# 策划总纲

本文是 `design/` 目录的入口与当前确定口径。分系统文档负责展开细节；若分系统之间出现冲突，以本文的“当前确定口径”为优先依据，再回写各分系统。

## 一句话定位

一款以修仙组织经营为主体、以灵气潮汐轮回为赛季节奏、以道统竞争和跨服窗口为长期目标的多人策略游戏。

## 核心体验

玩家不是扮演单个修士，而是经营一个修仙组织。组织通过地图扩张、资源采集、凡人城镇经营、灵气吸收、功法修炼、阵法流水线、法器配置和道统协作，在一次次灵气潮汐与轮回中积累优势。

每个 4 周小轮回都应形成清晰节奏：

| 阶段 | 玩家主行为 | 系统目的 |
|:----|:----------|:--------|
| 涨潮期 | 扩张、建设、采集、打通经脉 | 给玩家建立本轮计划的窗口 |
| 峰值期 | 突破、战争、秘境、跨服进攻 | 制造高收益高风险的爆发点 |
| 退潮期 | 巩固、收割、外交、调整阵法 | 让优势转化为稳定资产 |
| 谷底期 | 休整、内政、储备、推演下一轮 | 降低操作压力，准备下轮决策 |

## 当前确定口径

### 世界与服务器

- 世界由四界 / 四服组成。
- 每服独立运行地图和轮回，但遵守同一套潮汐规则。
- 每服潮汐周期为 4 周，四服相位错峰 1 周。
- 跨服交互围绕潮汐差形成战略窗口，不是全天候完全互通。

### 玩家主体

- 玩家主体是组织，不是单角色。
- 组织初始性质为门、派、教、宗之一。
- 组织性质影响资源路径、扩张方式、生存方式和成长天赋。
- “教”没有固定主城，依赖教义传播和信徒影响力维持势力范围。

### 灵气与修炼

- 灵气是全局核心资源，但不能只当作金币使用。
- 灵气链路为：外界灵气 -> 功法吸收 -> 经脉流转 -> 法术 / 阵法 / 法器输出。
- 关键效率包括吸取率、转化率、经脉连通度、利用率、储存衰减。
- 五行属性影响资源采集、功法适配、阵法效率、法器制造和战斗策略。

### 地图与资源

- 地图由区域 / 地块组成，包含灵山福地、凡人城镇、荒野、秘境、禁区。
- 凡人城镇提供人口、信仰、灵魂、税收和基础材料。
- 灵气节点提供高价值周期资源；五行精矿点提供阵法和法器材料。
- 占领更多地盘不应线性滚雪球，必须受到维护成本、经脉负载、外交压力和潮汐阶段制约。

### 阵法与法器

- 阵法是静态灵力流水线，负责生产、防御、传送、炼丹炼器、修炼加速等长期效果。
- 法器是动态灵力处理单元，可配置自动行为，也可接入阵法链。
- 阵法与法器的设计目标是让玩家持续优化“灵气输入 -> 材料加工 -> 战斗 / 生产输出”的自动化网络。

### 道统目标

三大道统是长期阵营与胜利路线：

| 道统 | 核心追求 | 玩法关键词 |
|:----|:---------|:----------|
| 司天 | 号令天道，安排定数 | 推演、预测、全服影响、资源调配 |
| 幽冥 | 往生转世，超脱轮回 | 灵魂、转世、战损恢复、低潮优势 |
| 真武 | 求真务实，以力证道 | 征服、战斗连胜、领地扩张、正面对抗 |

## MVP 建议范围

第一阶段不追求一次性完成所有设定，建议优先打通最小闭环：

1. 创建组织：支持门、派、教、宗的基础差异。
2. 基础地图：区域、凡人城镇、灵气节点、五行矿点。
3. 灵气收入：地盘灵气、聚灵阵、节点采集。
4. 灵气消耗：修炼、阵法维护、法器充能。
5. 修炼成长：功法品阶、五行属性、经脉节点、境界突破。
6. 阵法流水线：聚灵阵、炼器阵、护山大阵三类先行。
7. 领地争夺：基础占领、掠夺、防守结算。
8. 小轮回：4 周阶段、潮汐倍率、轮回奖励和灵气重置。
