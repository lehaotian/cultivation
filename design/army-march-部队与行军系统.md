# 部队与行军系统

## 核心理念：自由行军

部队不是"点击目的地→自动寻路→到达→战斗"的固定流程，而是**在地图上自由移动、实时可控**的活单位。行军途中可随时变向、布阵、交战、撤退。

```
自由行军 = 拖拽移动 + 实时指令 + 动态布阵
```

---

## 一、部队基础

### 部队构成

```
部队
├── 弟子（战斗单位）
│   ├── 修为境界
│   ├── 装备法器
│   └── 当前气血/灵力
├── 法器（装备槽）
│   ├── 攻击型（飞剑等）
│   ├── 防御型（法袍等）
│   └── 特殊型（阵盘等）
├── 阵法（行军阵型）
│   ├── 阵型模板
│   └── 当前展开状态
└── 辎重（物资）
    ├── 丹药
    ├── 灵石
    └── 材料
```

### 部队规模

| 规模 | 弟子数 | 法器槽位 | 阵型规模 | 移动速度 |
|:----|:-----:|:--------:|:--------:|:--------:|
| **小队** | 1~10 | 1 | 小 | 快 |
| **中队** | 10~50 | 3 | 中 | 中 |
| **大队** | 50~200 | 5 | 大 | 慢 |
| **军团** | 200~1000 | 10 | 超大 | 极慢 |

- 部队规模由组织等级+科技树解锁
- 弟子可来自组织成员（在线/离线均可被调遣）
- 每支部队需要至少1名"领队"（修为最高的弟子担任）

### 弟子配置

```
弟子在部队中的角色：
├── 领队 × 1          ← 决定部队整体属性
├── 阵眼 × 0~3        ← 维持阵法运转
├── 战斗组 × N         ← 常规作战
├── 法器师 × 0~5      ← 操控法器
└── 后勤 × 0~N        ← 携带辎重、治疗
```

配置界面类似编队系统：
1. 从组织弟子池中选择弟子
2. 分配角色（领队/阵眼/战斗/法器师/后勤）
3. 为弟子装配法器
4. 选择行军阵法
5. 确认出征

## 二、自由行军机制

### 行军状态

```
         ┌──────────┐
         │  待命    │ ← 在山门/据点内
         └────┬─────┘
              │ 出征
              ▼
         ┌──────────┐
    ┌───→│  行军    │ ← 在地图上移动中
    │    └────┬─────┘
    │         │ 遭遇/指令
    │         ▼
    │    ┌──────────┐
    │    │  布阵    │ ← 展开阵型（可移动/可静止）
    │    └────┬─────┘
    │         │ 接敌
    │         ▼
    │    ┌──────────┐
    │    │  战斗    │ ← 交战状态
    │    └────┬─────┘
    │         │
    │    ┌────┴─────┐
    │    │          │
    │ 撤退/战败   胜利
    │    │          │
    │    ▼          ▼
    │ 返回行军    打扫/继续
    └───────────────┘
```

### 行军操作

| 操作 | 说明 | 消耗 |
|:----|:-----|:----|
| **出征** | 部队从山门出发进入地图 | 无 |
| **移动** | 拖拽/点击目标位置，部队自由移动 | 灵力（持续） |
| **变向** | 行军途中随时更改目标点 | 无 |
| **急行军** | 消耗额外灵力加速移动 | 灵力×2 |
| **停止** | 部队原地待命 | 无 |
| **布阵** | 行军途中展开阵型 | 灵力（一次性） |
| **撤阵** | 收起阵型恢复行军状态 | 无 |
| **驻留** | 在资源点/城镇停留 | 无 |
| **回城** | 自动返回最近的山门/据点 | 灵力 |
| **解散** | 部队解散，弟子回归组织 | 有惩罚 |

### 移动方式

| 方式 | 速度倍率 | 灵力消耗 | 适用场景 |
|:----|:--------:|:--------:|:---------|
| **步行** | ×1.0 | 低 | 常规行军 |
| **御剑飞行** | ×2.0 | 中 | 快速移动 |
| **传送阵** | 瞬间 | 高 | 据点间快速转移 |
| **灵舟/飞梭** | ×3.0 | 极高 | 大规模部队长距离移动 |
| **遁术** | ×1.5 | 中 | 隐蔽移动（不显示在地图上） |

### 地图交互

- 部队在地图上以**旗帜/图标**形式可见（友方高亮，敌方模糊）
- 敌方部队的详细信息需要通过**侦察**获取
- 地形影响行军速度（山地-30%，水域-50%，平原+20%）
- 特殊地形触发特殊事件（秘境入口、遗迹等）

## 三、行军阵法

部队可以在行军途中展开阵法，这是自由行军的核心策略维度。

### 行军阵型

| 阵型 | 速度 | 防御 | 攻击 | 侦察 | 特点 |
|:----|:----:|:----:|:----:|:----:|:-----|
| **长蛇阵** ⬤⬤⬤⬤ | ★★★★ | ★★ | ★★ | ★★★ | 行军最快，适合赶路 |
| **方圆阵** ◯ | ★★ | ★★★★ | ★★ | ★★ | 防御最高，适合穿越危险区 |
| **锋矢阵** ▶ | ★★★ | ★★★ | ★★★★ | ★ | 攻击最高，适合突袭 |
| **鹤翼阵** 〰 | ★★ | ★★ | ★★★ | ★★★★★ | 侦察范围最大 |
| **雁行阵** V | ★★★ | ★★★ | ★★★ | ★★★ | 均衡，适合遭遇战 |
| **天覆阵** ★ | ★ | ★★★★★ | ★★★★★ | ★★★★★ | 高阶阵法，全面提升 |

### 布阵机制

```
布阵流程：
1. 部队行军途中下达"布阵"指令
2. 选择阵型模板（受阵法等级限制）
3. 阵眼弟子占据阵眼位置
4. 其他弟子按阵型自动排列
5. 法器嵌入阵法节点（可选强化）
6. 阵法展开完成，获得阵型加成

布阵耗时 = 基础时间 × (1 - 阵眼熟练度)
布阵消耗 = 阵型等级 × 单位灵力
```

### 行军+布阵组合策略

```
情景1：侦察到敌方部队 → 立即布阵（鹤翼阵） → 确认敌方阵容 → 切换锋矢阵 → 冲锋

情景2：被埋伏 → 方圆阵防御 → 边打边撤 → 退入友方领地

情景3：匀速行军（长蛇阵）→ 接近目的地 → 切换雁行阵 → 展开搜索

情景4：声东击西 → 一队假意布阵吸引注意 → 主力急行军绕后
```

## 四、法器在行军中的应用

### 可配置法器

行军/战斗过程中法器有不同的实用效果：

| 法器类型 | 行军效果 | 战斗效果 |
|:--------|:---------|:---------|
| **飞剑** | 探路（前方侦察） | 远程攻击 |
| **灵舟** | 提升行军速度 | - |
| **阵盘** | 快速布阵（缩短时间） | 阵型强度加成 |
| **灵珠** | 灵力储备（延长行军时间） | 灵力续航 |
| **隐环** | 隐蔽行军（不被侦察） | 偷袭加成 |
| **战鼓** | 提升士气 | 全属性加成 |
| **困敌索** | 拦截敌方部队 | 控制效果 |

### 法器配置策略

```
部队法器槽位有限，需要取舍：
- 追求速度 → 灵舟 + 轻装（少带法器）
- 追求战力 → 飞剑 + 阵盘 + 战鼓
- 追求隐蔽 → 隐环 + 少量弟子
- 追求持久 → 灵珠 + 后勤弟子
```

## 五、行军事件

行军途中会随机触发各种事件：

| 事件 | 概率 | 选项 |
|:----|:---:|:-----|
| **遭遇妖兽** | 中 | 战斗/绕路/驱赶 |
| **发现灵脉** | 低 | 采集/驻留标记/上报 |
| **遭遇敌方斥候** | 中 | 追击/隐蔽/绕路 |
| **发现秘境入口** | 极低 | 进入/标记/上报 |
| **弟子顿悟** | 低 | 停下修炼/继续行军 |
| **天象异变** | 低 | 找地方躲避/硬抗/布阵应对 |
| **散修投靠** | 低 | 招募/拒绝 |
| **辎重丢失** | 低 | 派人找回/放弃 |

## 六、视野与侦察

### 视野机制

```
友方部队视野：部队周围圆形区域（半径与阵型/法器相关）
敌方部队可见条件：
├── 进入我方的视野范围 → 可见
├── 被斥候/探路法器发现 → 可见
├── 处于战斗状态 → 可见（全地图通告）
└── 使用隐环/遁术 → 不可见

迷雾区域：未探索区域被迷雾遮挡
├── 部队经过后迷雾暂时散开
├── 离开后迷雾重新笼罩（但保留地形信息）
└── 高侦察阵型/法器可扩大视野范围
```

### 侦察对抗

| 行为 | 可被侦察 | 反制 |
|:----|:--------|:-----|
| 正常行军 | 是 | 无 |
| 隐环行军 | 否（但不可攻击） | 破隐法器 |
| 急行军 | 更容易被发现 | - |
| 夜间行军 | 更难被发现 | - |
| 布阵 | 暴露位置 | 撤阵后隐蔽 |
| 驻留 | 部分暴露 | 伪装 |

## 七、行军消耗

| 消耗项 | 计算方式 |
|:------|:---------|
| **灵力** | 弟子数 × 行军时间 × 地形系数 |
| **士气体力** | 行军时间越长下降越快，影响战力 |
| **丹药** | 长途行军自动消耗恢复丹药 |
| **法器耐久** | 法器在行军中使用会损耗 |

- 灵力耗尽：部队自动停下，无法继续行军
- 士气归零：部队可能溃散/叛逃
- 丹药不足：士气下降加速

## 八、多部队协同

玩家可同时控制多支部队，实现复杂战术：

```
分进合击：多路同时进攻，让敌方顾此失彼
诱敌深入：一队佯败，引敌入埋伏圈
围点打援：围住资源点，阻击援军
钳形攻势：两翼包抄
断后掩护：一队阻击追兵，主力撤退
```

多部队同时操作时，可编组为**集团军**统一指挥。
